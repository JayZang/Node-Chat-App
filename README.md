# Node-Chat-App
User can chat on the web which in the same room name.

## About Project
### skill
1. Node
2. WebSocket

### package
1. express
2. socket.io

## Usage
install packages
```
npm install
```
run test case
```
npm test
```
run App
```
npm start
```
Input url on your browser: 
[http://localhost:3000/](http://localhost:3000/)
